# Cybersecurity - FSI 2024/25

## About
Security challenges developed for the 
Computer Security (FSI) course at FEUP (Faculty of Engineering, 
University of Porto).

## Topics Covered
- SQL Injection
- XSS (Cross-Site Scripting) Attacks
- Buffer Overflows
- Network Security
- Cryptography
- Web Security

## Structure
Each LOGBOOK corresponds to a different security challenge, 
documenting the approach, findings and solutions.
